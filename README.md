# Job Analytics Dashboard (Tableau)

This project is developed as part of my Data Analyst Internship at ElevanceSkills. It focuses on analyzing job-related data using Tableau and creating interactive visualizations based on given conditions.

---

## Project Overview

The Job Analytics Dashboard provides insights into job postings by applying multiple filters and constraints. It includes various types of visualizations such as bar charts, scatter plots, tree maps, maps, stacked bar charts, and box plots.

The dashboard is designed to be interactive, allowing users to explore job data dynamically using filters.

---

## Dataset Description

The dataset contains job-related information such as job title, company name, location, salary range, experience, qualification, work type, and other attributes. This data is used to perform analysis based on task-specific conditions.
Due to large file size, the dataset is not included in this repository.

---

## Tasks Implemented

- Task 1: Preference vs Work Type (Bar Chart)  
- Task 2: Company Size vs Company Name (Scatter Plot)  
- Task 3: Top 10 Companies (Tree Map)  
- Task 4: Qualification Drilldown Map  
- Task 5: India vs Germany Comparison (Stacked Bar Chart)  
- Task 6: Work Type Salary Distribution (Box Plot)
